# Dev

1. Clonar el proyecto
2. Copiar el ```env.template``` y renombrar a ```.env```
3. Ejecutar
```
npm i
```
4. Levanta la imagen (Docker desktop)
```
docker-compose up -d
```
5. Levantar el Backend de Nest
```
npm run start:dev
```
6. Visitar apollo studio
```
localhost:3000/graphql
```
